Jenkinsfile
Declarative pipeline
  • Declarative syntax with stages{} block
  • GitHub webhook triggers on push to main
  • Environment variables for EC2 host/user
  • SSH credentials stored in Jenkins vault
☕
Build & Test
Maven 3.9 + JDK 17
  • mvn clean package generates fat JAR
  • JUnit 5 unit & integration tests
  • Test reports archived as Jenkins artifacts
  • Failure stops pipeline immediately
🚀
Deploy Stage
SSH + nohup
  • SCP copies JAR to EC2 /opt/app/
  • Kills existing process by PID file
  • Starts new instance with nohup in background
  • Health check confirms app is live on port
🔔
Notifications
Post pipeline hooks
  • post { always, success, failure } blocks
  • Build status reported back to GitHub
  • Console log tailing for debug
  • Workspace cleanup on completion

Monitoring with Prometheus & Grafana

Full observability from system metrics to application health — Node Exporter, Prometheus, and Grafana all running on the same EC2 instance.

📡
Node Exporter
Exposes EC2 system metrics — CPU, memory, disk I/O, network — on port 9100 for Prometheus to scrape every 15 seconds.
🔥
Prometheus
Scrapes Node Exporter + Spring Actuator metrics. Stores time-series data, evaluates alert rules, serves on port 9090.
📊
Grafana
Dashboards for JVM heap, HTTP request rate, error rate, and host system health. Pre-built Spring Boot + Node panels on :3000.
💓
Spring Actuator
Exposes /actuator/prometheus endpoint. JVM, HTTP, DB pool, and custom app metrics scraped directly by Prometheus.
